Overview

Purpose

This site is meant to be a hub for a set of beginner steps on learning a number of coding languages. I plan on giving a number of basic guides on python and C# but others could be included as well. Some extra tips will also be provided to learn some things related to certain fields as well, such as making games.

Audience

This site is mainly aimed at those who are new to starting out to coding, and those who have an idea on what they want to work on specifically.

Dynamic elements

The Dynamic elements will be used to hide and show the various langauges and their respective langagues. If one selects python, then, it'll show things related to python, and the same for C#, or perhaps another language in the future. I think dynamic elements can also be used to check code, such as have we practiced in certain activities.
